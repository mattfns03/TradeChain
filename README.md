# TradeChain

TradeChain is a blockchain-based trade finance platform designed to streamline international trade using smart contracts, escrow settlement, decentralized identity, oracle-based shipment verification, and zk-KYC.

## Tech Stack

- Solidity
- Hardhat
- Next.js
- FastAPI
- PostgreSQL
- IPFS
- Chainlink
- Docker

## Architecture

Importer → Escrow → Shipment Verification → Automated Settlement
